География олимпиады

За время проведения олимпиады в ней приняли участие более 320 тысяч школьников из 85 субъектов РФ и стран СНГ.

История

  • 2004-2006

    • Событие: Первая Всероссийская викторина по финансовому рынку для старшеклассников.

    Период проведения: Февраль 2004 – июль 2006.

    Участники: 500 учащихся 9-11 классов из 30 субъектов РФ.

    Организатор: Институт фондового рынка и управления.

  • 2006-2007

    • Событие: Вторая Всероссийская викторина по финансовому рынку для старшеклассников.

    Период проведения: Февраль 2006 – февраль 2007.

    Участники: 420 учащихся 9-11 классов из крупных городов России (Иркутск, Новосибирск, Красноярск, Омск, Екатеринбург, Самара, Москва, Санкт-Петербург).

    Организатор: Институт фондового рынка и управления.

  • 2007-2008

    • Событие: Третья Всероссийская викторина по финансовому рынку для старшеклассников.

    Период проведения: Ноябрь 2007 – октябрь 2008.

    Участники: 4079 учащихся из 70 субъектов РФ.

    Организатор: Институт фондового рынка и управления.

  • 2008-2009

    • Событие: Четвертая Всероссийская викторина по финансовому рынку для старшеклассников.

    Период проведения: 2008–2009 учебный год.

    Участники: 1726 учащихся из 67 субъектов РФ.

    Особенности: Викторина включена в Перечень олимпиад школьников на 2008–2009 учебный год (1 уровень).

    Организатор: Федеральная служба по финансовым рынкам, Институт фондового рынка и управления.

  • 2009-2010

    • Событие: Пятая Всероссийская викторина по финансовому рынку для старшеклассников.

    Период проведения: 2009–2010 учебный год.

    Участники: 3973 учащихся из 77 субъектов РФ.

    Особенности: Викторина включена в Перечень олимпиад школьников на 2009–2010 учебный год (1 уровень).

    Организатор: Федеральная служба по финансовым рынкам, Институт фондового рынка и управления.

  • 2010-2011

    • Событие: Шестая Всероссийская викторина по финансовому рынку для старшеклассников.

    Период проведения: 2010–2011 учебный год.

    Участники: 3166 учащихся из 81 субъекта РФ.

    Организатор: Федеральная служба по финансовым рынкам, Институт фондового рынка и управления.
